# Google Play Store Analysis
This project focuses on analyzing a dataset from the Google Play Store to uncover insights about app trends, user preferences, and factors that influence app success.
Using Python-based data analysis techniques, this notebook explores app categories, ratings, installs, pricing strategies, and more to identify meaningful patterns in the mobile app ecosystem.

## Objectives
### 1. Understand the distribution of apps across various categories
### 2. Analyze rating trends and identify what affects higher ratings
### 3. Explore pricing strategies (free vs paid apps)
### 4. Identify factors influencing app installs
### 5. Detect anomalies and clean inconsistent data
### 6. Visualize key insights for better interpretation
